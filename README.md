# 📊 Social Media Analytics Using Excel

## 🧾 Project Overview
This project focuses on analyzing social media data using Microsoft Excel to extract meaningful insights about user engagement, content performance, and audience behavior. It leverages Excel’s built-in tools such as Pivot Tables, Charts, Conditional Formatting, and basic formulas to transform raw data into actionable insights.

---

## 🎯 Objectives
- Analyze engagement metrics (likes, shares, comments)
- Identify top-performing posts and content types
- Understand audience behavior trends over time
- Measure campaign effectiveness
- Build interactive dashboards for reporting

---

## 📂 Dataset Description
The dataset typically includes:
- Post ID
- Date & Time
- Platform (Instagram, Twitter, Facebook, etc.)
- Content Type (Image, Video, Text)
- Likes
- Comments
- Shares
- Reach / Impressions
- Hashtags used

---

## 🛠️ Tools & Features Used in Excel
- **Pivot Tables**: For summarizing engagement metrics
- **Charts (Bar, Line, Pie)**: For visual analysis
- **Conditional Formatting**: Highlight top/bottom performers
- **Formulas**:
  - `SUM()`, `AVERAGE()`, `COUNT()`
  - `IF()` for classification
  - `VLOOKUP()` / `XLOOKUP()` for data mapping
- **Slicers & Filters**: For interactive dashboards
